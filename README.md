
# SheerID Telegram Bot (Infrastructure Scaffold)

This repository provides **infrastructure only**:
- Telegram bot (webhook)
- Plans, credits, analytics
- Admin dashboard
- Auto-update hook
- Fly.io deployment

⚠️ **No verification automation logic is included.**
Plug your own compliant integrations where marked.

## Deploy (Fly.io)
- Set secrets: BOT_TOKEN, DATABASE_URL, ADMIN_KEY
- fly deploy
